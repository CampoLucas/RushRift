# Audio Manager
**Type:** `class` | **Inherits from:** `MonoBehaviour` | **Implemented in:** `Game`
### Description
Manages the playback of audio in the game using pooled `AudioSources`.

### Serialized Variables
| Variables | Description                                           |
| --------- | ----------------------------------------------------- |
| `sounds`  | List of available sounds configured in the inspector. |
| `mixer`   | The mixer the sounds use.                             |
### Public Static Methods
| Methods | Description                                              |
| ------- | -------------------------------------------------------- |
| `Play`  | Plays a sound by name if it exists in the audio manager. |
